# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目性质

这不是代码仓库，而是小说《寻找安托万》（作者：玉草央）的润色项目。原稿为 `antoine.md`（引子+30章，约13.7万字符），故事档案存于 `summary.md`。

## 润色工作流程

用户要求按以下顺序逐章进行，不可跳过，不可一次性输出大量内容：

1. 先读 `summary.md` 了解故事全貌（人物、设定、章节地图、文风特点）。
2. 每次处理一章：先给出整体评价 → 拆成6-8个部分 → 每部分标注【保留/改写/压缩/其他】→ 逐部分列【原文】+【建议改写】，由用户做最终决定。
3. 用户确认后，将改动写入 `antoine.md`，再进入下一章。
4. 所有对话使用中文。

## 润色目标

达到可出版水准。保持作者的文风：第一人称回忆口吻、黑色幽默自嘲、大量自然/海洋/烟火比喻、中法双语穿插、超自然情节冷处理、母题意象呼应。

## 润色中需要注意的规则

- 每次对话不要给出太多内容，逐步推进。
- 必须严格按照六到八个部分、逐部分标注、原文+建议改写的格式。
- 需要实质性地审视每部分：是否需要删减、是否句子过长、是否有重复信息、是否需要扩写情绪或场景，而不是简单地说"保留"。
- 减少破折号的使用，用户认为 AI 味很浓且写法老旧。
- 可用交互式 HTML widget 展示对比内容，但这不是必须的。
- 笔误修正（错别字、语病）可以直接列出，用户确认后一次性写入。
